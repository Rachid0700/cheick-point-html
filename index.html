A propos

Nous sommes ravi de vous voir consulter notre page sachez que vous êtes venu au bon endroit. The univers Of Programming est une structure de formations du developpement Web et elle a plus de 5 ans d'expériences dont le Fondateur L'ivoirien Ibrahim Abdoul Rachid Diomandé. Cette structure forme en Front-End et Back-End avec les langages tels que: HTML, CSS Javascrit et React Js ainsi que d'autres langages de programation. Comme vous le voyez cela vous permettra à devenir soit Développeur Front-End, Back-End ou Full Stuck cela selon vos objectifs. Pour les débutants qui n'ont aucune notion sur ces langages Veuillez trouver dans les articles ci-dessous quelques bèves informations liées à ces différents langages.

HTML

HTML signifie « HyperText Markup Language » qu'on peut traduire par « langage de balises pour l'hypertexte ». Il est utilisé afin de créer et de représenter le contenu d'une page web et sa structure. Il est le corps ou encore le squellette de notre site Web.

CSS

Le CSS correspond à un langage informatique permettant de mettre en forme des pages web (HTML ou XML). Il est permets de styliser une page Web: les couleurs, la taille, forme et bien d'autres... Ce langage est fondamentale et impératif à l'utilisation

Javascript

JavaScript est un langage de programmation utilisé par les développeurs pour concevoir des sites web interactifs. Les fonctions JavaScript peuvent permettre d'améliorer l'expérience utilisateur d'un site web. il a un rôle suprême du CSS.

React Js

Un avantage majeur de l'utilisation de ReactJS pour les projets web est la possibilité de réutiliser les composants. Cela signifie que les développeurs Web n'ont pas à écrire de code à partir de zéro. Il est un Frame Work du Javascript ou disons une amelioration du JavaScript.
